- [ ] Page titles are descriptive

- [ ] No sensitive data in code

### Configuration
- [ ] Update base URL in robots.ts
- [ ] Update base URL in sitemap.ts
- [ ] Configure custom domain (if applicable)
- [ ] Set up SSL certificate
- [ ] Configure redirects (if needed)
